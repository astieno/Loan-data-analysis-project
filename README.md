# Loan Data Analysis Project

## Overview

This project involved analysing a loan dataset containing 3,000 records to identify patterns, relationships, and trends relevant to lending and financial decision-making.

The analysis was conducted using KNIME and focused on exploring key attributes through data visualisation and exploratory data analysis techniques.

## Tools Used

- KNIME

## Skills Demonstrated

- Data cleaning and preparation
- Exploratory data analysis (EDA)
- Data visualisation
- Correlation and trend analysis
- Business insight generation
- Report writing and communication

## Project Activities

- Analysed a dataset containing 3,000 loan records.
- Created visualisations including scatter plots and line charts to investigate relationships between attributes.
- Identified trends and patterns within the dataset to support data-driven conclusions.
- Communicated findings through a structured analytical report.

## Key Learning Outcomes

- Using KNIME workflows to analyse structured datasets.
- Interpreting visualisations to uncover business insights.
- Evaluating relationships between financial and demographic attributes.
- Presenting analytical findings in a clear and professional format.

## Note

The dataset and KNIME workflow file are not publicly available due to university academic integrity requirements.
